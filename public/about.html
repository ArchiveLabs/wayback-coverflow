Wayback Time Machine

The Wayback Time Machine provides a way to visually explore the evolution of a website.

It is powered by the Internet Archive's Wayback Machine. See the Wayback Machine CDX API.

Created by Jono Brandel, Richard Caceres and Linda Peng at the 2017 Internet Archive Experiments Hackathon. See source code on Github.
